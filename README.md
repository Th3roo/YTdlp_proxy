# YTdlp_proxy
# Прокси-сервис для потоковой передачи видео

Это асинхронный веб-сервис на FastAPI, который позволяет стримить видео с YouTube, обеспечивая поддержку HTTP Range Requests для перемотки.

## Ключевые возможности

-   Потоковая передача видео по ID.
-   Поддержка перемотки (seeking) благодаря обработке заголовка `Range`.
-   Использование cookies браузера (Firefox) для доступа к приватным видео.
-   Локальное кэширование только запрашиваемых частей видео.
-   Автоматическое удаление временных файлов.

## Установка

1.  Клонируйте репозиторий:
    ```bash
    git clone <your-repo-url>
    cd <repo-folder>
    ```

2.  Создайте и активируйте виртуальное окружение (рекомендуется):
    ```bash
    python -m venv venv
    source venv/bin/activate  # Для Windows: venv\Scripts\activate
    ```

3.  Установите зависимости:
    ```bash
    pip install -r requirements.txt
    ```
    *Примечание: для использования cookies вам должен быть установлен Firefox.*

## Запуск

Запустите сервер с помощью Uvicorn:

```bash
uvicorn main:app --reload
```

Сервис будет доступен по адресу `http://127.0.0.1:8000`. Документация OpenAPI (Swagger UI) будет доступна по адресу `http://127.0.0.1:8000/docs`.

## Использование

### Просмотр в браузере
Откройте в браузере (например, VLC или веб-плеере) URL вида:

`http://127.0.0.1:8000/stream/dQw4w9WgXcQ`
(Замените `dQw4w9WgXcQ` на ID нужного видео).

### Использование с `curl`

**Получить первые 1024 байта:**
```bash
curl --range 0-1023 http://127.0.0.1:8000/stream/dQw4w9WgXcQ -o part1.mp4
```

**Получить следующие байты:**
```bash
curl --range 1024-2047 http://127.0.0.1:8000/stream/dQw4w9WgXcQ -o part2.mp4
```
